# Collateral Damage

## Overview
Collateral Damage is a game where the player takes on the role of a cleanup crew worker in a world dominated by a powerful hero. Instead of fighting enemies, the player must deal with the destruction left behind by battles.

## Gameplay
- Clean up debris scattered around the environment
- Repair damaged objects
- Avoid dangerous hazards such as fire and explosions
- Earn points by restoring the environment

## Controls
- WASD / Arrow Keys → Move
- E / Interact Key → Clean / Repair

## Objective
Restore the environment as efficiently as possible while avoiding hazards created by the unseen hero.
